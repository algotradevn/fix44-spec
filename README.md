# FIX 4.4 Trading API Specification

FIX 4.4 is the global standard for electronic order routing — but in Vietnam, few brokers have adopted it. [**Algotrade**](https://algotrade.vn) bridges that gap end-to-end: spec, client development, joint testing, and production operation for equities and derivatives.

This repo is our **spec template** for new broker integrations — covering session management, order entry, execution reporting, lifecycle examples, and Vietnamese market specifics. Replace `<Broker>` with the target broker's name and work through the Open Items section together.
